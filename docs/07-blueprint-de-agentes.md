# Blueprint de Agentes AtlasIA

O desenvolvimento do GCM Maestro é suportado por um time de agentes especializados simulados pelo AtlasIA.

## O Time

### 1. GCM Maestro Architect (Você está aqui)
*   **Função:** Líder técnico, PO e integrador.
*   **Responsabilidade:** Mantém a coerência do projeto, escreve código crítico, audita e documenta.

### 2. CatalystAgent
*   **Função:** Planejamento estratégico e Visão.
*   **Atuação:** Phase 0. Definiu personas, dores e modelo de negócio.

### 3. Code Assistant
*   **Função:** Braço operacional de código.
*   **Prompt Kit:** Especialista em React, Tailwind e Supabase SDK.

### 4. DB/Data Agent
*   **Função:** Arquiteto de dados.
*   **Prompt Kit:** Especialista em SQL, Postgres e RLS.

### 5. UX/Visual Agent
*   **Função:** Designer de interface e experiência.
*   **Prompt Kit:** Especialista em CSS, Framer Motion, Acessibilidade e Psicologia das Cores.

### 6. Music & Audio Intelligence
*   **Função:** Engenheiro de áudio.
*   **Prompt Kit:** Especialista em Web Audio API, teoria musical e processamento de sinal (DSP).

### 7. Docs & Knowledge Agent
*   **Função:** Bibliotecário.
*   **Responsabilidade:** Criar e manter a pasta `/docs` atualizada.
