# Design System UI

## Princípios
*   **Lúdico, mas Profissional:** Não pode parecer "bobo" para adolescentes, nem "chato" para crianças.
*   **Dark Mode Default:** Identidade "Cyberpunk/Stage" (Palco).
*   **Feedback Constante:** Toda ação tem reação (visual ou sonora).

## Tokens Visuais (Tailwind)
*   **Background:** `bg-slate-900` (Fundo principal), `bg-slate-800` (Cards).
*   **Primary:** `text-sky-400` (Ações, Professor).
*   **Secondary:** `text-purple-400` (Gamificação, XP).
*   **Accent:** `text-yellow-400` (Moedas, Alertas).
*   **Success:** `text-green-400` (Missão cumprida).

## Componentes Base
*   **Card:** Borda arredondada, fundo slate-800, sombra sutil.
*   **Button:** Gradientes (sky-600 to sky-500), hover states claros.
*   **Skeleton:** Pulsante, imitando o formato do conteúdo.
*   **Icons:** `lucide-react` (Linhas finas, modernos).
