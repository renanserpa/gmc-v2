# Glossário

| Termo | Definição |
| :--- | :--- |
| **GCM Maestro** | Módulo do professor para gestão de turmas e conteúdo. |
| **OlieMusic** | Nome da plataforma/ecossistema. |
| **XP (Experience Points)** | Pontos de experiência ganhos por atividades. Definem o Nível. |
| **Olie Coins** | Moeda virtual para compra de itens cosméticos na Loja. |
| **Streak (Fogo)** | Contador de dias consecutivos de prática/uso do sistema. |
| **RLS (Row Level Security)** | Política de segurança do banco de dados que restringe acesso linha a linha. |
| **HUD (Heads-Up Display)** | Painel superior do aluno com resumo de status (Nível, XP, Moedas). |
| **Mission** | Tarefa atribuída pelo professor (ex: "Tocar exercício X"). |
| **Elefante/Passarinho** | Metodologia de ensino para diferenciar sons Graves e Agudos. |
